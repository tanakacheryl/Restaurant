<h1>Restaurant Management Web App</h1>

<h2>Description</h2>
This project consists of a web-based restaurant management and digital menu system developed using PHP, MySQL, Bootstrap, JavaScript, HTML, CSS, and XAMPP. The application allows restaurant administrators to manage food categories, menu items, customer orders, and daily special offers through a dynamic and responsive interface.

The system includes both a customer-facing website and an admin panel where administrators can add, update, search, and delete menu items and promotional offers. Customers can browse food categories, search for meals, place orders, and view restaurant information such as services, reservations, and contact details.

The application integrates a MySQL database managed through phpMyAdmin to store menu data, offers, orders, and user information. PHP is used for backend logic and database communication, while Bootstrap and custom CSS provide a modern and responsive user interface across different devices..
<br />


<h2>Languages / Tech Stack Used</h2>

- <b>Frontend: HTML, CSS, JavaScript, Bootstrap</b> 
- <b>Backend: PHP</b>
- <b>Database: MySQL (phpMyAdmin)</b>
- <b>Server: XAMPP</b>
